Project Objective

This project analyses retail sales data to identify profit leaks, meaning regions and product categories where the company is losing money despite high sales volume. The goal is to provide actionable insights that help regional managers optimise their discount strategies and shipping logistics.

Business Impact


Automated financial auditing: flags transactions where the profit margin falls below the break even threshold, so issues surface before they compound
Discount strategy optimisation: a performance matrix visualises the relationship between discount rate and net profit, exposing the point where discounts start destroying value
Root cause analysis: a category drill down isolates specific underperforming sub categories, such as high volume items with high return rates, supporting targeted inventory decisions
Operational scalability: environment agnostic configuration means the pipeline can move to a cloud PostgreSQL instance on Azure or AWS without code changes


Tech Stack


Language: Python 3.x
Database: PostgreSQL, connected through SQLAlchemy and Psycopg2
Data processing: Pandas, NumPy
Visualisation: Seaborn, Matplotlib
Configuration: Python Dotenv for secure environment variable management


What the Pipeline Does


Ingests raw transactional data from Sales_SuperStore.csv
Normalises inconsistent column headers into a clean, consistent format
Standardises date fields into a uniform format
Loads the cleansed data into a PostgreSQL database using SQLAlchemy
Computes profit, margin, and discount metrics broken down by region and category
Generates the visualisations saved in the images folder, including the discount performance matrix and category level profit charts

How to Run


Clone the repository
Install dependencies using pip install -r requirements.txt
Create a .env file with your PostgreSQL connection string, this file is excluded from version control through .gitignore
Run Analysis.py to execute the full pipeline, from data loading through to chart generation
View the generated charts in the images folder
